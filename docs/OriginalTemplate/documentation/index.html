Installing Template

  • After unzip Theme Files, you will found HTML Template folder with all this files you will upload to your site.
  • This template can be seen in any web broswer you from your computer or laptop and you do not need to upload the files into a server because.
  • Start your FTP client (I persoanlly use Filezila which is free) and login. Login to your server. You will need your server name (often times this will be your domain name), username, and password that you setup with the hosting company.
  • Locate the domain folder you wish to upload your Theme files to on your server. You are going to want to click and drag every file that you have labeled .html, .css, .js, images into your FTP client.
  • Once the files are done uploading, rename the file you want to be your home page as 'index.html'.
  • Is should be like: www.domainname.com/index.html

JavaScript

This theme imports three Javascript files.

  1. bootstrap.min.js - bootstrap framework
  2. plugins.js - plugins for console
  3. jquery.cycle2.min.js - cycle2 plugin for slider
  4. jquery.cycle2.swipe.js - cycle2 plugin for mobile
  5. ios6fix.js - cycle2 plugin to fix ios 6
  6. jquery.isotope.min.js - isotope filter plugin
  7. scrollTo.min.js - scrollto plugin
  8. fresco.js - lightbox media gallery plugin
  9. jquery.debouncedresize.js - smartresize plugin
  10. jquery.throttledresize.js - smartresize plugin
  11. main.js - My own javascript
